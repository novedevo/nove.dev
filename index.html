About Me

I'm a computer engineering student located in Vancouver, BC and studying at Simon Fraser University. My interests include music, hiking, coding and video games. Python + VS Code + WSL2 = development joy. In my spare time, I can sometimes be found solving puzzles such as nonograms, shikaku, and crosswords. My passion projects include game design and development using Phaser, home network architecture in the Unifi ecosystem, and home server architecture under FreeBSD (soon to be Debian) with TrueNAS. I'm currently taking a course on data structures and algorithms using Java, and I have experience in C, C++, Python, HTML/CSS/JS, and hardware design using VHDL and Quartus.
